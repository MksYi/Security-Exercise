## 解題方法
在原始碼頁面中，發現線索，如下。  

```
<!--  /web-serveur/ch6/phpbb -->
```

接這又進入到一片空白的冏境，但該題目的名稱為 `Install Files`，顯然跟 phpbb 的安裝有關係，在 URL 加上 `install`，解決該題。
